# 1、功能介绍
通过python设置企业微信群聊机器人每天定时发送提醒消息并@所有人<BR/>
可以实现的功能：<BR/>
a) 每日可按设定的时间定时发送消息并@所有人<BR/>
b) 可设置不同时间发送不同消息<BR/>
c) 可以设置不同的重复时间间隔<BR/>
